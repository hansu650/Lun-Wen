# vision transformer
- 把图片变成序列
- 把图片分成小块，用处理文本的方式来处理图片

## 解决的问题-cnn
- 归纳偏置(模型被迫对数据进行了假设)图像有局部性和平移不变性
- 感受野限制 只能看到小窗口(堆叠到后面才能全局)
- 缺乏全局，没办法很远的地方链接全局
- 可迁移性弱

# 核心思想
1. 切成补丁 patch
- pxp  16x16(建议大小)
- 224x224
- 得到14x14x3=196x3



2. 线性投影
- 16x16x通道数(3-rgb)
- 展平 16x16x3=768
- 全连接(线性投射)用E学习   768映射到固定D，可以放到后面的向量

- 最前面类似bert人为的加了一个可学习向量class token
- 作为全局信息收集器
- 得到 (196+1)xD 个向量 patch embedding

3. 加入位置编码
- 空间关系不能丢失
- 我们2丢失了位置信息

## 创建一个位置编码向量
- N+1xD  一样可学习矩阵
- 一样第0行给cls token 后面顺着给196个图片
- 位置编码向量直接加到patch embedding

4. 送到transformer
- 每个块都可以获得所有全局信息
- 分类取cls
