# transformer
- attention

- 先把位置编码加进词向量

- wq
- wk
- wv

- 可以多个qkv(头)权重矩阵
- 然后再把多个结果拼接

- q和k相乘得到权重，再去和v相乘累加就是合并了所有的上下文信息的单词了

- Q：我当前这个 token 想找什么信息
- K：每个 token 能提供什么“被匹配的线索”
- V：每个 token 真正携带、最后要被加权汇总的内容
